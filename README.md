# 🌟 SC Huang | Personal Portal & Ambient Greeting Page

A beautiful, premium, and highly responsive personal landing portal designed specifically for **SC Huang**. This web application features modern glassmorphism, dynamic time-of-day greetings, a customizable clock widget, particle background animations, and interactive theme aura rotation.

🚀 **Live Demo:** [https://shengchieh2121.github.io/2026.05.29/](https://shengchieh2121.github.io/2026.05.29/)

---

## ✨ Features & Highlights

- 🔮 **Glassmorphism Design Aesthetics**: Crafted with harmonized color palettes, blur effects (`backdrop-filter`), smooth shadows, and micro-interactions that elevate the desktop or mobile viewing experience.
- 🕒 **Dynamic Time-of-Day Greetings**: Automatically greets SC Huang based on the exact time of day (e.g., *Good morning*, *Good afternoon*, *Good evening*, *Good night*) with custom positive subtitles for each state.
- ⏱️ **Interactive Digital Clock**: Real-time ticking clock with a custom-styled live status dot. Users can **click the clock card** to toggle between **12-hour (AM/PM) and 24-hour** display formats.
- 🎨 **Ambient Aura Cycling**: Click the **Cycle Theme Aura** button to dynamically shift the glowing ambient blobs, matching neon gradient titles, and outer cards to distinct states:
  - *Original Aurora* (Indigo / Pink / Cyan)
  - *Sunset Fire* (Amber / Red / Pink)
  - *Forest Neo* (Emerald / Cyan / Blue)
  - *Cyber Cosmic* (Violet / Pink / Blue)
- ✨ **Spark Inspiration Widget**: A curated compilation of motivational quotes with premium transition animations, providing daily sparks of wisdom with a single click.
- 🌊 **Background Interactive Particle Overlay**: An elegant custom HTML5 Canvas particle system that flows continuously and responds seamlessly to screen resizing.

---

## 🛠️ Technology Stack

- **Markup:** Semantic HTML5
- **Styling:** Vanilla CSS3 featuring extensive CSS Variables for seamless state transitioning, Outfit & Plus Jakarta Sans typography from Google Fonts
- **Interactions:** Vanilla Modern JavaScript (ES6+)
- **Background Effects:** HTML5 Canvas API

## 📦 How to Run Locally

Since this project is fully client-side and requires zero external build tools, you can run it instantly:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ShengChieh2121/2026.05.29.git
   cd 2026.05.29
   ```
2. **Open the web page:**
   - Double-click `index.html` to open it directly in any modern web browser.
   - Or, run it via a local development server (such as Live Server in VS Code) for the best local debugging experience.
